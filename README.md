# Roy Andrej Martinsen — Foto

Editorial fotoportefølje. Bygget for hånd med HTML, CSS og en liten JS-fil.

## Lokalt
Åpne `index.html` direkte, eller server:
```
python3 -m http.server 8000
```

## Struktur
```
index.html
css/style.css
js/main.js
images/
  portrett/  utvalgte/  dokumentar/  drone/
  prosjekt/  konsept/   mat/         wes-anderson/
```

## Deploy
Repoet er koblet til Vercel via GitHub. Push til `main` → auto-deploy.
